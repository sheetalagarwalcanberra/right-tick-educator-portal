# Right Tick Educator Onboarding Portal

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal, usually:

http://localhost:5173/

## Build for production

```bash
npm run build
```

## Main file

`src/App.jsx` contains the full React single-page portal prototype:
- Educator onboarding wizard
- Contract acceptance flow
- Required document upload UI
- Admin dashboard
- Compliance status badges
